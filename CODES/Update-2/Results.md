# Features #


## 1) $x^{-1}$ ##

### Usage ###

Calculates the inverse. Say x=2, so it will give you the decimal value of $\frac{1}{2}=0.5$

**Step-1**: *Input the value of x=2*


**Step-2**: Click the operator $x^{-1}$
**Step-3**: Value Displayed
